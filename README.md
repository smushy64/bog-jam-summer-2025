# Bog Jam Summer 2025

Written in C++ using [raylib](https://www.raylib.com/).

## Build Steps
- build cbuild

```bash
clang cbuild.c -o cbuild
```

- build project

```bash
./cbuild build -release
```

- project will be in ./build directory
